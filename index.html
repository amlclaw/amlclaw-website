How It Works

🔍

Screen

Screen blockchain addresses against compliance rules. Supports Tron, Ethereum, Bitcoin, Solana. Traces up to 5 hops of fund flow.

⚙️

Rules

Generate machine-readable AML detection rules from 40+ international regulations. Manual input, document analysis, or web search.

📋

Policy

Create formal compliance policy documents from your ruleset. Professional templates for Singapore MAS, Hong Kong SFC, Dubai VARA.

Quick Start

# Clone and install
git clone https://github.com/amlclaw/amlclaw.git
pip install requests python-dotenv

# Load a default ruleset
cp amlclaw/defaults/rulesets/singapore_mas.json ./rules.json

# Screen an address
python3 amlclaw/scripts/run_screening.py Tron <ADDRESS> --scenario deposit
